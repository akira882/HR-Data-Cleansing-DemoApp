# ビジネス・インパクト分析 (Business Impact Analysis)

本プロジェクトが実務に導入された際、組織にどのような「定量・定性的メリット」をもたらすかを分析しました。

## 1. 定量メリット: コスト削減と効率化
人事アナリスト（月給40万円と仮定）が、手作業で行っている分析業務を本システムでリプレイスした場合の試算です。

| 項目 | 従来の手作業 | 本システム導入後 | 削減率 |
| :--- | :--- | :--- | :--- |
| データ修正・クレンジング | 約4時間 | **3秒 (自動)** | 99.9% |
| KPI集計・グラフ作成 | 約2時間 | **1秒 (自動)** | 99.9% |
| 異常値の再確認・修正依頼 | 約2時間 | **1分 (検知のみ)** | 90% |
| 経営報告用PDF作成 | 約2時間 | **5秒 (自動)** | 99.9% |
| **合計 (月次1回あたり)** | **10時間** | **約2分** | **約99% 削減** |

- **年間削減時間**: 120時間 / 人
- **コスト価値**: 年間約 **30〜40万円相当** の人件費を「戦略的業務（面談、制度設計など）」へ転換可能。

## 2. 定性メリット: データの信頼性とガバナンス
- **人為的ミスの排除**: Excelの関数エラーやコピー＆ペーストミスによる報告のバイアスを排除。
- **意思決定の均質化**: 誰が担当しても同じクリーニング基準、同じKPI定義でレポートが生成されるため、年度や拠点を超えた正確な比較が可能。
- **早期リスク発見**: AIによる異常検知により、「不満が溜まっている部署」や「データ入力のミス」を放置することなく、即座に修正・介入が可能。

---
*本分析は、中規模組織（従業員500名程度）の人事部門をモデルとしています。*
