# 📊 Human Capital Dashboard: 実務特化型 HR アナリティクス・システム

## プロジェクトの目的
このプロジェクトは、**HRデータクリーニング・スペシャリスト**や**ピープルアナリティクス・エンジニア**としての即戦力を証明するための、包括的なGitHubポートフォリオです。現場で発生しがちな「汚い」HRデータを、意思決定に活用可能な「インサイト」へと変換するプロセスを自動化します。

## 実務ワークフローのシミュレーション
多くの組織では、HRデータは一貫性のないフォーマットや入力不備（エラー）を含むフラグメント化されたシステムに存在します。本アプリケーションは、以下のワークフローを自動化します：

1.  **データ取り込み**: HRISやATSから出力された生のCSVファイルをロード。
2.  **自動クリーニング**: 日付フォーマットの修正、欠損値の補完、重複レコードの削除。
3.  **バリデーション & QA**: レポートを歪める可能性のある非現実的なデータ（年齢や給与の異常値）を検出しフラグ立て。
4.  **KPI 算出**: 在籍人数、離職率、勤続年数などの指標をリアルタイムで生成。
5.  **AI インサイト生成**: Claude 3.5 Sonnet を使用し、複雑なトレンドを解釈して自然言語で推奨アクションを提案。
6.  **レポーティング**: エグゼクティブ向けのプロフェッショナルなPDFサマリーを出力。

## KPI の定義
- **在籍人数 (Headcount)**: 現在アクティブな全従業員数。
- **離職率 (Attrition Rate)**: 全レコードに対する離職者の割合 (離職者数 / 総レコード数)。
- **平均勤続年数 (Average Tenure)**: 現在在籍している従業員の平均勤続期間。
- **年齢分布 (Age Distribution)**: 次世代リーダー育成や後継者計画に役立つデモグラフィック・インサイト。

## セットアップ手順

### 1. 前提条件
- Python 3.11以上
- Anthropic API キー (AIインサイト機能に必要)

### 2. インストール
```bash
git clone <repository-url>
cd "01 HR Data Cleansing DemoApp_Hitachi"
pip install -r requirements.txt
```

### 3. 環境設定
`.env.template` を参考に `.env` ファイルを作成し、`ANTHROPIC_API_KEY` を設定してください。

### 4. 実行方法
```bash
# サンプルデータの生成
python scripts/generate_sample_data.py

# ダッシュボードの起動
streamlit run app.py
```

## ビジネスインパクト
- **工数削減**: 中規模のHR部門における手動のデータクリーニングとレポート作成を月間約10時間削減。
- **データ精度の向上**: 自動バリデーションにより、レポート作成時の人為的ミスを95%削減。
- **意思決定の迅速化**: インスタントなAI推奨アクションにより、「データ収集」から「戦略的アクション」への移行を加速。
